# テストJSON 一覧

ファイル名の接頭辞で期待表示が分かる：
- **`正常_`** … 正しく描画される（崩れたら不具合）
- **`異常_`** … 壊れた/境界の入力。**表示は degraded（崩れて当然）だがクラッシュしないこと**を確認するもの

## 正常系
| ファイル | 確認内容 |
|---|---|
| 正常_旧フォーマット単一.json | 旧 `{project,tasks}` 後方互換 |
| 正常_複数4プロジェクト.json | 複数プロジェクト・帯・全体1本イナズマ |
| 正常_大量タスク120.json | スケール（120リーフ）・折りたたみ |
| 正常_エッジ多様.json | 全完了/全未着手/フラット/同日/遅め・早め着手の進行中（イナズマ点は本日線）/長文 |
| 正常_広い期間1年.json | 1年スパン・多数の月ヘッダ・横スクロール |
| 正常_空プロジェクト.json | `projects:[]`（空表示で正しい） |
| 正常_空タスク.json | タスク0のプロジェクト |
| 正常_特殊文字エスケープ.json | `<script>` 等が**エスケープされ実行されない**こと |
| 正常_4階層ネスト.json | 4階層（色はL3止まり・崩れない） |
| 正常_5階層.json | 5階層（4階層超でも色はL3でクランプ・崩れない） |
| 正常_カスタムキー.json | `_ai`/`_money`/`_links`/`_meta` 等の `_` 始まりキーが**描画で無視され画面に出ない**こと |
| 正常_備考URL.json | 備考の `http(s)://` が**自動リンク化**・`javascript:` は無効・クエリの `&` が壊れない |
| 正常_終了遅延.json | 終了遅延=予定終了日へ左突出／着手遅延（未着手）／境界・早め着手・完了の確認（本日2026-06-10前提） |

## 異常系（表示は崩れる・クラッシュしないことを確認）
| ファイル | わざと壊した内容 / 期待表示 |
|---|---|
| 異常_欠損フィールド.json | qty/hours/plan/actual 欠損 → 工数空・バー無し・「—」 |
| 異常_不正な日付.json | 終了<開始の逆転・実績未来・実績逆転 → バー無し/異常バー |
| 異常_数値境界.json | qty0/少数/hours999 → 工数0・1.5人日・超ワイドバー |
| 異常_壊れMS重複ID.json | 範囲外/色なし/不正日付MS・重複ID（折りたたみキー衝突） |
| 異常_color注入.json | MSの `color` に属性脱出・`javascript:`・閉じタグ注入 → `isColor()` が弾き**注入されない**（`#hex`のみ許可） |
| 異常_非オブジェクトトップ.json | トップが非オブジェクト（数値）→ 空表示でクラッシュしない（null/配列/文字列はno-crashテストで併検） |

## E2E テスト（自動）

`e2e/` に headless Chromium（Playwright）の自動テストを同梱（実行方法は [`e2e/README.md`](e2e/README.md)）。
上記の JSON データは目視確認だけでなく **`test_corpus.py` が全件を読み込み JSエラー/NaN無しを機械検証**する（graceful degradation の回帰）。

| テスト | 検証内容 |
|---|---|
| test_corpus | 全fixture＋非オブジェクトtop（null/配列/数値/文字列）で no-crash / no-NaN |
| test_display | 全列のセル表示値・派生計算（工数=qty×h÷8・親集計）・進行中/—/✓・linkify・カスタムキー非表示 |
| test_save | 各フィールドの編集が**正しいJSONパス**に保存（field→path 混線検出） |
| test_security | XSSエスケープ（esc）・color属性インジェクション防御（isColor） |
| test_render | イナズマ点x座標が仕様式どおり・2段見出し・NaN無し |
| test_gantt_bars | 予実バー種別（予定帯/実績/終了遅延over/着手遅れgap）・「+N」ラベル値・titleの正確値（#65/#34） |
| test_highlight | ガント十字ハイライト：行/日付列/左表データ列の display 切替・pointer-events:none（#48） |
| test_done | 先頭✓トグルで actual.end＝本日・未着手なら start も補完（#37） |
| test_date_shorthand | 日付ショートハンド入力・適応型短縮表示・年補完・範囲外無視（#59） |
| test_edit / test_dates / test_permissions | 編集モード・日付欄・FSA保存パス |
| test_collapse / test_i18n / test_ui / test_load / test_links | 折りたたみ（ヘッダ▼▶＋Ctrl+Z）・言語・保存状態・読込経路・URLリンク |
